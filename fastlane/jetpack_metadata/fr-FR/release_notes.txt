Si votre site utilise un thème reposant sur des blocs, vous pouvez désormais modifier la page d’accueil de votre site directement à partir de la liste des pages. Émoticône Mains levées.

Nous avons résolu un problème concernant l’ouverture des liens dans un nouvel onglet. Lorsque vous saisissez du texte après ce lien, vous pouvez désormais annuler et rétablir le texte sans aucun problème.

Les paramètres de détails visibles sont désormais désactivés pour les vidéos VideoPress qui n’appartiennent pas à votre site.

La fonction Appuyer pour modifier est disponible pour tous les types de blocs imbriqués : vous n’avez plus à parcourir tous les niveaux d’imbrication pour atteindre un élément de contenu spécifique.

Nous avons corrigé un problème concernant l’imbrication qui entraînait la suppression de tout texte imbriqué dans un élément de liste lors de sa fusion dans un paragraphe.

Lors de la suppression de bugs, nous avons également résolu quelques problèmes dans le lecteur. Lorsque vous filtrez les sujets que vous suivez, les textes en langues non latines (comme l’arabe) s’affichent désormais correctement. Les actions Rapport et Bloc sont également de nouveau disponibles dans le menu Article.

Enfin, nous avons modifié le design des cellules désactivées pour déterminer plus facilement quand une cellule n’est pas utilisée.
