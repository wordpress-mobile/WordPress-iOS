Google 도메인을 더 쉽게 이전할 수 있도록 도메인 포커스 카드를 알림판에 추가했습니다. (여러 사이트에 이 카드를 표시하고 싶지 않으신가요? "더 보기" 버튼을 눌러서 숨기세요.)

블록 편집기에도 몇 가지 변경 사항을 적용했습니다.

- 이제는 선택하면 확인할 수 있도록 개요가 소셜 링크 블록에 포함되어 있습니다.
- 사용자 정의 글꼴 크기와 스타일이 적절히 표시되지 않았던 iOS 사용자의 문제를 해결했습니다.
- HTML 편집기에 더는 실행 취소 버튼과 다시 실행 버튼이 표시되지 않습니다.
- 이제는 커버 블록 색상 선택기를 사용할 때 선택한 색상의 16진수 코드가 표시됩니다.

마지막으로, 리더에서 새 사이트를 팔로우할 때 앱이 충돌하는 원인이 될 수 있었던 버그도 수정했습니다.
