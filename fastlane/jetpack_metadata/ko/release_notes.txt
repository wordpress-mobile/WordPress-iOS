사이트에서 블록 기반 테마를 사용하는 경우 이제는 페이지 목록에서 바로 사이트의 홈 페이지를 편집할 수 있습니다. 손을 들고 있는 이모티콘입니다.

링크가 새 탭에서 열리는 설정 문제를 해결했습니다. 이 링크 뒤에 텍스트를 입력할 때 이제는 문제 없이 실행을 취소하고 다시 실행할 수 있습니다.

사이트에 속하지 않는 비디오프레스 비디오에 표시되는 상세 정보 설정이 이제는 비활성화되었습니다.

이제는 모든 중첩된 블록 유형에 눌러서 편집을 사용할 수 있습니다. 더는 특정 콘텐츠 부분에 도달하려고 모든 중첩 레벨을 눌러보지 않아도 됩니다.

중첩의 경우 목록 아이템을 문단에 병합하면 해당 목록 아이템의 중첩된 텍스트가 모두 제거되는 버그를 수정했습니다.

어떻게든 버그를 없애려고 리더에서도 몇 가지를 계속 수정했습니다. 팔로우한 게시글을 필터링할 때 라틴어가 아닌 언어(예: 아랍어)의 텍스트가 이제는 올바르게 표시됩니다. 신고 및 차단 조치도 글 메뉴의 원래 위치로 돌아갑니다.

마지막으로, 셀이 사용 중이지 않을 때 더 쉽게 알 수 있도록 비활성화한 셀의 디자인도 업데이트했습니다.
