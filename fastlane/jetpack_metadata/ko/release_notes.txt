이제는 블록을 끌어다 놓을 수 있어서 콘텐츠를 마음대로 쉽게 재정렬할 수 있습니다. 계속 발전하겠습니다.

블록 편집기에서 변경된 것은 또 있습니다. 어둡게 모드에서 작업할 때 텍스트를 더 쉽게 읽을 수 있도록 색상 대비를 조정했습니다. 갤러리 블록의 링크 연결 대상 설정 문제도 해결하고, 해당 블록에 이미지 크기 설정도 추가했습니다.

이제는 지원되지 않는 블록 편집기에서 워드프레스닷컴 둘러보기 배너 팝업 창이 표시되지 않습니다. 둘러보기에서는 이제 새 사이트를 제작하는지 또는 기존 사이트로 작업하는지에 따라 다른 퀵 스타트 작업 세트가 표시됩니다. 알림 확인 및 미디어 업로드와 같은 새 작업을 아주 쉽게 익힐 수 있습니다.

마지막으로, 앱을 더 쉽게 사용할 수 있도록 상당히 성가신 버그를 없앴습니다.

- 오른쪽에서 왼쪽으로 쓰는 언어를 사용하는 기기에서 이제는 위젯 텍스트가 뒤집히지 않습니다.
- 통계 인사이트와 다른 사이트 섹션 사이에서 전환할 때 앱이 충돌하지 않습니다. 더 있습니다.
- 글 목록에서 생성 화면이 이제는 분할 화면으로 작업하는 iPad 사용자에게 표시됩니다.
- 사이트 제목을 사이트 설정에서 변경하면 내 사이트에서도 업데이트됩니다.
- 사용 가능한 미디어가 없을 때 적합한 메시지가 미디어 선택기에 표시됩니다.
- 기기에서 큰 글꼴을 사용할 때 이제는 사이트 생성 프로세스의 탐색 버튼 크기가 적절히 조정됩니다.
