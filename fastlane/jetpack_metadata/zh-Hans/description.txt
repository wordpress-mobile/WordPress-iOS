适用于 WordPress 的 Jetpack

随时随地发布 Web 版。 Jetpack 可用于创建网站，且功能繁多！

创建

给您的重要创意在网上找到安家之地。 Jetpack iOS 版由 WordPress 支持，可用于构建网站和创作博客。 使用它创建您的网站。
从丰富的 WordPress 主题中挑选合适的外观和风格，然后借助照片、颜色和字体加以定制，打造出您独有的网站。
内置的快速入门贴士将指导您掌握基础的网站创建知识，帮助您建立走向成功的新网站。 （我们不仅提供网站创建工具，还充当您的合作伙伴和啦啦队！）

分析与见解

实时查看您网站的统计数据，随时了解站点上的动态。
通过研究每日、每周、每月和每年的数据分析，跟踪哪些文章和页面在一段时间内获得最多流量。
使用流量图了解访客来自哪些国家/地区。

通知

获得有关评论、点赞和新粉丝的通知，从而实时了解人们对您的网站有何反应。
及时回复新评论，保持对话通常，并答谢读者。

发布

借助编辑器撰写最新动态、故事、照片随笔、公告等！
使用相机和相册中的照片和视频来生动呈现文章和页面，或者通过应用程序内置的专业摄影图片库（免费使用）找到最理想的图片。
将创意保存为草稿，并在灵感再现时回头审视这些草稿，或者安排要在日后发布的新文章，让您的站点始终保持新鲜感和吸引力。
添加标签和类别，帮助新读者发现您的文章，同时观察读者群体的增长情况。

安全和性能工具

如果出现错误，可以从任意位置恢复您的站点。
扫描威胁，轻触一下即可解决问题。
密切关注站点动态，查看谁更改了什么内容以及更改时间。

阅读器

Jetpack 的功能不仅限于创作博客，它还能让您在 WordPress.com 阅读器中与一群作者建立联系。 根据标签来浏览数千个主题、发现新的作者和组织，以及关注激发您兴趣的人士。
使用“保存以备后用”功能留存吸引您的文章。

共享

设置自动分享功能，以便当您发表新文章时在社交媒体上告知您的粉丝。 文章会自动交叉发表到 Facebook、Twitter 等社交媒体。
在文章中添加社交共享按钮，让访客可以与其关系网共享文章，让您的粉丝成为您的传播大使。
